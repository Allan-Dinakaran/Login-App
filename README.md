# 📱 Jetpack Compose Login App

This project is a basic Android login application built using **Kotlin** and **Jetpack Compose**.  
It is created as a practical implementation of the concepts I have learned in modern Android development.

---

## 🚀 About the Project

This app focuses on building a simple and clean login UI while applying core Jetpack Compose concepts.

---

## 🧩 Features

- `Column` for structured layout  
- `OutlinedTextField` for email and password input  
- `Toast` messages for user feedback  
- Proper alignment and spacing of UI components  
- Basic function handling for login action
